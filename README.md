## How to Run Locally
- Open `index.html` in a browser (use VS Code Live Server for best experience).
- For tests: `npm install` then `npm test`.

## Keywords
budget tool

## Features Implemented
- Add/edit/delete expenses with form
- Persistence via LocalStorage
- Totals by category and time periods
- Anomaly detection and highlighting
- Chart.js visualization of trends
- Unit tests with Vitest
